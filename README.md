# Simple-network-technology-calculator-software

简易的网络技术计算器软件，集成了多种网络技术相关的计算工具，适用于网络学习、运维及技术人员日常工作需求。

本软件已申请通过中国版权保护中心计算机软件著作权登记，请在合法范围内使用。

## 功能说明

### 1. IP地址计算器
- 支持十进制/二进制IP地址和子网掩码输入
- 自动转换IP地址与子网掩码的十进制和二进制格式
- 计算并展示以下网络信息（均包含十进制和二进制两种格式）：
  - 网络地址
  - 广播地址
  - 地址类别（A类、B类、C类等）
  - 第一个可用IP地址
  - 最后一个可用IP地址
  - 受限广播地址
  - 主机号

### 2. 内存单位转换器
- 支持B（字节）、KB（千字节）、MB（兆字节）、GB（吉字节）、TB（太字节）之间的单位转换
- 提供两种转换基数选择：
  - 二进制（1024进制）
  - 十进制（1000进制）
- 输入任意单位数值后，点击"转换"按钮可自动计算并显示其他单位的对应值

### 3. 二进制与十进制转换器
- 支持十进制数转二进制数
- 支持8位二进制数转十进制数
- 提供输入验证功能，确保二进制数为8位且仅包含0和1
- 可通过"清除输入"按钮重置转换状态
- 支持回车键快速触发计算

### 4. 停机时间与系统可用性计算器
- 支持输入年、月、日、小时、分钟格式的停机时间
- 自动计算并判断系统可用性等级：
  - 停机时间≤5分钟：系统可用性达到99.999%
  - 停机时间≤53分钟：系统可用性达到99.99%
  - 停机时间≤8.8小时：系统可用性达到99.9%
  - 超过上述时间：系统可用性低于99.9%
- 提供输入清除功能

### 5. ITU标准数据传输速度计算器
- 基于ITU标准，计算OC-X（Optical Carrier）光载波等级的传输速度
- 输入OC-X中的X值，自动计算对应的数据传输速度（单位：Mbps）
- 计算原理：传输速度 = X × 51.84 Mbps

### 6. IPv6地址验证工具
- 支持输入IPv6地址并验证其有效性
- 对于无效地址，显示具体的错误原因
- 结果展示包含"是否有效"状态和详细判断理由

### 7. 交换机全双工带宽计算器
- 支持输入不同速率端口的数量：
  - 100Mbps端口数量
  - 1000Mbps端口数量
  - 10000Mbps端口数量
- 自动计算交换机全双工模式下的总带宽
- 结果根据数值大小自动转换为Mbps或Gbps单位展示

### 8. 子网划分计算器
- 支持基于IP地址和CIDR前缀进行子网划分
- 可指定子网数量和每个子网所需的主机数量
- 支持按"从大到小"或其他顺序排序子网
- 自动计算满足主机数量需求的子网划分方案

## 使用方法
1. 运行`ncre_main.py`文件启动软件
2. 通过软件内的功能导航选择所需工具
3. 在对应功能界面输入参数，点击"计算"或"转换"按钮获取结果
4. 可通过"清除输入"按钮重置当前功能的输入状态
